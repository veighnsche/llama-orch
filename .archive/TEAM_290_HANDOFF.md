## How It Works

### Local Operations (localhost)

```bash
# Install hive locally
./rbee hive install
# ‚Üí Auto-detects binary from target/
# ‚Üí Installs to ~/.local/bin/rbee-hive
# ‚Üí No SSH required

# Start hive locally
./rbee hive start
# ‚Üí Spawns process directly (no SSH)
# ‚Üí Uses tokio::process::Command
# ‚Üí Verifies with pgrep

# Stop hive locally
./rbee hive stop
# ‚Üí Uses pkill (no SSH)
# ‚Üí Verifies with pgrep
```

### Remote Operations (SSH)

```bash
# Install hive remotely
./rbee hive install --host gpu-server --binary ./rbee-hive
# ‚Üí Connects via SSH (uses ~/.ssh/config)
# ‚Üí Uploads binary
# ‚Üí Installs to /usr/local/bin/rbee-hive

# Start hive remotely
./rbee hive start --host gpu-server
# ‚Üí Connects via SSH
# ‚Üí Runs nohup rbee-hive --port 9000 &
# ‚Üí Verifies with pgrep

# Stop hive remotely
./rbee hive stop --host gpu-server
# ‚Üí Connects via SSH
# ‚Üí Runs pkill -f rbee-hive
# ‚Üí Verifies with pgrep
```

## SSH Configuration

### User Setup

**1. Add host to ~/.ssh/config:**
```ssh
Host gpu-server
  HostName 192.168.1.100
  User ubuntu
  IdentityFile ~/.ssh/id_rsa
  Port 22
```

**2. Copy public key to remote:**
```bash
ssh-copy-id gpu-server
```

**3. Use rbee-keeper:**
```bash
# Install hive remotely
./rbee hive install --host gpu-server

# Start hive remotely
./rbee hive start --host gpu-server
```

### Advanced SSH Features

**ProxyJump:**
```ssh
Host gpu-server
  HostName 192.168.1.100
  User ubuntu
  ProxyJump bastion-host
```

**SSH Agent:**
```bash
eval $(ssh-agent)
ssh-add ~/.ssh/id_rsa
./rbee hive install --host gpu-server  # Uses agent
```
