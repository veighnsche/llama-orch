# Shared Components

This page demonstrates the shared UI components from `@rbee/ui` that maintain design consistency across the rbee platform.

## Design Tokens

All components use the rbee design system with:
- **Primary Color**: Honeycomb Yellow/Gold (#f59e0b)
- **Dark Mode**: Automatic theme switching
- **Consistent Spacing**: 4px base unit

## Usage in Documentation

These components can be used throughout the documentation to:
- Highlight important actions with **Buttons**
- Mark status or categories with **Badges**
- Group related content with **Cards**
