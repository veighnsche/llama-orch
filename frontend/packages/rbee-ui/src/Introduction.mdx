# rbee UI Component Library

Welcome to the **rbee UI** component library! This is the shared design system for all rbee applications, ensuring consistent user experience across the platform.

## Design Philosophy

The rbee design system is built around:

- **Honeycomb Yellow/Gold** (#f59e0b) - Our primary brand color inspired by bees
- **Professional yet approachable** - Enterprise-grade but not corporate
- **Data-focused** - Clear information hierarchy
- **Privacy-first** - Security and trust communicated through design

## Component Categories

### Atoms
Small, single-purpose components:
- **Button** - Primary interactive elements
- **Badge** - Status and category indicators

### Molecules
Composed components with specific functions:
- **Card** - Content grouping with header, content, and footer

### Coming Soon
More components will be added as they're extracted from the commercial site or needed by user-docs.

## Usage

Import components from their respective categories:

```tsx
import { Button, Badge } from '@rbee/ui/atoms';
import { Card, CardHeader, CardTitle, CardContent } from '@rbee/ui/molecules';
```

Import design tokens:

```tsx
import { colors, spacing, radius } from '@rbee/ui/tokens';
```

Import shared styles:

```tsx
import '@rbee/ui/styles';
```
