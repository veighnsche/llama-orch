# Welcome to rbee Documentation

**rbee** is a private LLM hosting platform built for the Netherlands, providing secure, compliant, and high-performance AI infrastructure.

## What is rbee?

rbee enables organizations to run large language models on their own infrastructure or through our managed service, ensuring data sovereignty and GDPR compliance.

## Key Features

- **Private & Secure**: Your data never leaves your infrastructure
- **GDPR Compliant**: Built for European data regulations
- **High Performance**: Optimized for GPU acceleration
- **Easy to Deploy**: Simple setup and configuration
